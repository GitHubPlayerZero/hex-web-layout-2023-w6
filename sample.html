background-img 測試

結果：background-img 在 local 測試的路徑不管有沒有加 assets 都會顯示，但放上 GitHub Pages 就一定要正確的路徑。
註：路徑是視 CSS 寫在哪裡而定，若是寫在 HTML 上，路徑會是 ../assets/images/；若是寫在 SCSS 裡，因為統一由 all.scss import 進來，路徑會是 ../images/。
此範例是寫在 _sample.scss，統一由 all.scss 載入。

img src="../assets/images/event/01.jpg"
